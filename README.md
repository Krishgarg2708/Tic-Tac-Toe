#  Tic Tac Toe Game (Python + Tkinter)

A simple yet fun **Tic Tac Toe (X and O)** game built using **Python and Tkinter GUI**.  
It supports **two-player mode**, automatic **win/draw detection**, and a clean **graphical interface**.

---

##  Features

-  Two-player gameplay (X vs O)
-  Automatic turn switching
-  Win, draw, and reset detection
-  Minimal and colorful Tkinter GUI
-  Replay option without restarting the app

---

##  Tech Stack

- **Language:** Python  
- **GUI Framework:** Tkinter  
- **Version:** Python 3.8 or above
